4. Diccionario de datos¶

En esta instancia se detallan las caracteristicas lógicas de cada elemento que forma parte de todo flujo de datos del sitema, a cada uno de estos elemtos se le llaman campos que sera la información que almacenara el sistema en una base de datos.

Si una persona es tiene como t_funcionaro = Administrativo, es decir, el usuario esta llenado un formulario para agregar un Administrativo poseerá los además los siguientes campos propios:

Campo Tipo Descripción
cargo_adminis choice
Indica el cargo del administrativo
con las siguientes opciones:
  • Auxiliar administrativo
  • Auxiliar de servicios generales
  • Celador
  • Profesional especializado
  • Profesional universitario
  • Secretario
  • Técnico aministrativo

Si una persona es tiene como t_funcionaro = Directivo, es decir, el usuario esta llenado un formulario para agregar un Directivo poseerá los además los siguientes campos propios:

Campo Tipo Descripción
cargo_directiv choice
Indica el cargo del directivo
con las siguientes opciones:
  • Coordinador
  • Director rural
  • Director de nucleo
  • Rector
  • Supervisor

Si una persona es tiene como t_funcionaro = Docente, es decir, el usuario esta llenado un formulario para agregar un Docente poseerá los además los siguientes campos propios:

Campo Tipo Descripción

Campo Tipo Descripción
